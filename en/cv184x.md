## 概述 {#overview}

CV184xH-P 是一款提供数据安全、生态链开放的深度学习视觉处理器，具备 8M 专业安防影像品质和 1.5TOPS@INT8 算力。

甲辰计划 CV184x 合作项目的所有开源资源均托管在 [GitHub](https://github.com/rv2036/cv184x) 上发布。

## 产品特点 {#features}

### 基础性能

| 项目         | 规格                                      |
| ------------ | ----------------------------------------- |
| 峰值性能     | 1.5 TOPS@INT8                             |
| 视频编码器   | H.264 & H.265                             |
| 专业安防影像 | 支持 8M、三目影像处理、宽动态、超星光降噪 |
| 处理器       | 双核 CPU                                  |

### 核心特性

- 自主知识产权 张量处理器
- 专业安防影像
- 完整的深度学习工具链
- 安全处理器架构
- 高整合高效能平台

## 性能参数 {#parameters}

| 参数类别       | 具体规格                                      |
| -------------- | --------------------------------------------- |
| **Processor**  | CA53@1.1GHz& C906@800MHz                      |
| **TPU**        | INT8@1.5TOPS                                  |
| **DDR**        | SiP DDR                                       |
| **VI**         | 3 video input 支持 MIPI、Sub-LVDS、HiSPi 接口 |
| **Audio**      | ADC/DAC/I2S                                   |
| **Encode**     | H.265/H.264/MJPEG 9Mp25[16:9 or 4:3]          |
| **Ethernet**   | 10/100 RMII+10/100 MAC PHY                    |
| **ISP**        | ISP-v4.0，支持最高 8Mp30 的基础 ISP 处理      |
| **Peripheral** | SD/USB 2.0/I2C/PWM/RTC/UART/SPI/GPIO          |
| **Package**    | BGA 10x10                                     |
