## Overview {#overview}

The CV184xH-P is a deep learning vision processor designed for data security and open ecosystem chains. It delivers 8M professional-grade security image quality and 1.5 TOPS @ INT8 computing power.

All open-source resources for the Jiachen Project CV184x collaborative project are hosted and released on [GitHub](https://github.com/rv2036/cv184x).

## Features {#features}

### Performance Base

| Item                 | Specification                                                                 |
| -------------------- | ----------------------------------------------------------------------------- |
| Peak Performance     | 1.5 TOPS @ INT8                                                               |
| Video Encoder        | H.264 & H.265                                                                 |
| Professional Imaging | Supports 8M, Triple-lens processing, WDR, and Super Starlight noise reduction |
| Processor            | Dual-core CPU                                                                 |

### Core Characteristics

- Proprietary Tensor Processing Unit (TPU)
- Professional Security Imaging
- Comprehensive Deep Learning Toolchain
- Secure Processor Architecture
- Highly Integrated High-Performance Platform

## Specifications {#parameters}

| Category       | Detailed Specifications                                       |
| -------------- | ------------------------------------------------------------- |
| **Processor**  | CA53@1.1GHz & C906@800MHz                                     |
| **TPU**        | INT8@1.5TOPS                                                  |
| **DDR**        | SiP DDR                                                       |
| **VI**         | 3 Video Inputs; Supports MIPI, Sub-LVDS, and HiSPi interfaces |
| **Audio**      | ADC/DAC/I2S                                                   |
| **Encode**     | H.265/H.264/MJPEG 9Mp25 [16:9 or 4:3]                         |
| **Ethernet**   | 10/100 RMII + 10/100 MAC PHY                                  |
| **ISP**        | ISP-v4.0; Supports basic ISP processing up to 8Mp30           |
| **Peripheral** | SD/USB 2.0/I2C/PWM/RTC/UART/SPI/GPIO                          |
| **Package**    | BGA 10x10                                                     |
